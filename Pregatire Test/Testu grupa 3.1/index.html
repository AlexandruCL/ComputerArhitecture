Working procedure

Create a pesonal directory, of the form "FirstName_LastName". Solve the proposed problems in your personal folder. For each problem, download the associated archive, extract it and change the working directory in Modelsim (using "File -> Change Directory ..." menu) to the problem's extracted folder. Solve the problem by completing the Verilog implementation in the indicated source file.

For testing an implementation execute, in the Transcript window of the Modelsim environment, command do test. The provided test script indicates possible failed configurations for correcting your solution.

After solving the proposed problems, archive your personal folder, including all solutions. Upload your solution's archive on the dedicated Virtual Campus test activity.
